STAY IN THE LOOP

Stay In The Loop is a puzzle-platformer in which you are trapped in a time loop. You must spot the one NPC repeating their actions before the timer runs out, collapsing the loop.

Solo Project, 2024 GMTK Game Jam Software: Game Maker Studio 2, Paint.NET, Audacity Roles: Game Designer, Artist, Programmer, Level Designer, UI Designer Unique Systems: NPC Decision Making, Movement System

PROJECT DETAILS

Stay In The Loop is an infinite high-score game where each loop loads a random preset map with NPCs, their number increasing each loop. NPCs use three behavior types: preset, random calm, and random energetic. With one always following a looping, predefined moveset. The minimalist UI keeps focus on tracking NPC movements.

LEVEL DESIGN

Levels are designed to accommodate many NPCs while allowing the player to move easily and locate the target.
